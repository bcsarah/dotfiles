# Dotfiles

This repository contains my **dotfiles** since February 2026, focused on a modern, minimal, and consistent environment using the **Gruvbox theme** across the entire system, running **i3wm**.

---
## 🎨 Theme

Uses **Gruvbox**, with an aesthetic that is:
- Easy on the eyes  
- Focused on visual comfort  
- Consistent across all tools

## 🧰 Tools

- **WM:** i3wm
- **Terminal:** kitty
- **Shell:** zsh
- **Editor:** vim
- **Bar:** i3status
- **Launcher:** dmenu
- **File Manager:** thunar
- **Fetch:** fastfetch
- **Git UI:** lazygit
- **System:** archlinux

## ✨ Customization

You can easily tweak:
- Gruvbox color tones  
- i3 binds and settings
- Vim plugins
